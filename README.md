## 🖥️ Active Directory Domain Lab (company.local)

### Overview
Built and configured a Windows Server domain environment using virtualization to simulate a real-world IT infrastructure.

---

### Environment
- Windows Server 2019 (Domain Controller)
- Windows 10 Client Machine
- Oracle VirtualBox
- Domain: company.local

---

### Key Configurations
- Promoted server to Domain Controller
- Configured Active Directory Domain Services (AD DS)
- Joined Windows 10 client to domain
- Created and managed user accounts
- Configured organizational units (OUs)

---

### Skills Demonstrated
- Active Directory user management
- Domain joining and authentication
- Group policy basics
- Troubleshooting domain connectivity issues
